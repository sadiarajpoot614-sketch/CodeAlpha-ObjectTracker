# CodeAlpha-ObjectTracker

An AI-powered real-time object detection and tracking system using YOLOv8 + DeepSORT.  
Built for CodeAlpha AI Internship - Task 2

## Features
- Real-time object detection from webcam and video files
- Assign unique ID to each tracked object
- Uses YOLOv8 for detection + DeepSORT for tracking
- Bounding boxes with class name and tracking ID

## Tech Stack
- **Python**
- **YOLOv8** - Ultralytics
- **OpenCV**
- **DeepSORT-Realtime**

## Installation
Install the required libraries:
```bash
pip install ultralytics opencv-python deep-sort-realtime
